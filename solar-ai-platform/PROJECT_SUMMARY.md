# 📊 SolarScan 프로젝트 요약

## 🎯 프로젝트 개요

**프로젝트명**: SolarScan - AI 기반 태양광 설치 최적화 플랫폼

**목적**: 경기도 기후플랫폼(climate.gg.go.kr) 데이터를 활용하여 주택 구매 전 태양광 설치 적합성과 예상 비용 절감액을 AI로 분석하는 웹 서비스

**개발 기간**: 6개월 (MVP 3개월 + 정식 출시 3개월)

**예상 투자액**: 1억 3천만원

## 💡 핵심 가치 제안

1. **사전 확인**: 부동산 방문 전 온라인으로 태양광 설치 가능성 확인
2. **정확한 예측**: AI 기반 발전량 및 비용 절감액 예측 (95%+ 정확도)
3. **빠른 분석**: 주소 입력 후 30초 이내 결과 제공
4. **데이터 신뢰성**: 경기도 공공 데이터 기반 객관적 분석

## 🎨 주요 기능

### 1. 주소 기반 분석
- **입력**: 경기도 내 주소
- **출력**: 
  - 지붕 분석 (면적, 방향, 경사각)
  - 월별/연간 발전량 예측
  - 설치 비용 및 보조금 계산
  - 투자 회수 기간 (ROI)
  - CO2 감축량 등 환경 기여도

### 2. 비교 분석
- 최대 5개 주소 동시 비교
- 일사량, 발전량, 절감액 비교 차트
- 최적 위치 추천

### 3. 히트맵
- 경기도 전역 일사량 히트맵
- 지역별 태양광 효율 순위
- 인터랙티브 지도 인터페이스

### 4. 프리미엄 리포트
- 상세 PDF 리포트 (₩29,000)
- 20년 시뮬레이션
- 최적 설치 용량 추천
- 업체별 견적 비교

## 🛠 기술 스택

### Frontend
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State**: Zustand
- **Charts**: Recharts
- **Maps**: Kakao Maps API

### Backend
- **Framework**: FastAPI
- **Language**: Python 3.11+
- **Database**: PostgreSQL + PostGIS
- **Cache**: Redis
- **Queue**: Celery + RabbitMQ
- **ML**: XGBoost, TensorFlow
- **CV**: OpenCV

### AI/ML
- **발전량 예측**: XGBoost Regressor (RMSE < 5%)
- **지붕 분석**: Computer Vision (Mask R-CNN)
- **시계열 예측**: LSTM (선택적)

### Infrastructure
- **Cloud**: AWS (EC2, RDS, S3, CloudFront)
- **Frontend Hosting**: Vercel
- **Container**: Docker, Docker Compose
- **CI/CD**: GitHub Actions
- **Monitoring**: Sentry, CloudWatch

## 📈 시장 분석

### 타겟 시장
- **경기도 가구 수**: 약 540만 가구
- **잠재 시장**: 단독주택 + 연립/다세대 = 140만 호
- **목표 시장**: 140만 호 × 10% = **14만 호**

### 경쟁 환경
| 경쟁사 | 강점 | 약점 |
|--------|------|------|
| 한국에너지공단 햇빛지도 | 공공 신뢰도 | UI/UX 구식, 상세 분석 부족 |
| 민간 태양광 업체 | 설치 연계 | 객관성 부족, 영업 목적 |
| Google Project Sunroof | 기술력 우수 | 한국 데이터 부족 |

**차별점**: 경기도 특화 + AI 정밀 분석 + 실시간 비용 계산 + 직관적 UX
